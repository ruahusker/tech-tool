## TechKit script catalog (macos)

Run with: bash "__TECHKIT__/macos/<script>" [args]
Refer to a tool by its Name (wrap in **double asterisks** in analysis to make it clickable). Full details: __CATALOG__.

- System Report (system_report.sh) — Model/chip/serial, macOS version, Rosetta, uptime, memory+swap pressure, storage summary, battery, reboot history. Use for: first look at any Mac; need specs/serial; general health check.
- Disk Health (disk_health.sh) — SMART status, APFS container usage, snapshot count; --verify runs live read-only volume check. Use for: slow Mac; disk full mysteries; suspected drive failure. | args: --verify
- Resource Usage (top_processes.sh) — Top CPU/memory processes, memory pressure, swap, hung (I/O-stuck) processes, thermal throttling. Use for: Mac is slow; fans loud; beachballs. | args: top_n
- Startup Programs (startup_items.sh) — Third-party LaunchDaemons/Agents with target binaries, login items, BTM dump, running non-Apple jobs, cron. Use for: slow login; popups; suspected adware/persistence.
- Network Diagnostics (network_diagnostics.sh) — Layered check: interfaces, gateway, internet by IP, DNS, captive-portal/HTTP, DNS config, proxy, Wi-Fi, VPN tunnels. Use for: no internet; Wi-Fi connected but nothing loads; only some sites work.
- Connectivity Test (test_connectivity.sh) — Targeted host test: DNS (system + direct), ping, TCP port via nc, optional traceroute. Use for: can't reach a specific server/printer/share. | args: host* port --trace
- Crash & Log Summary (log_summary.sh) — Crash/hang/panic reports (7d) with per-app frequency, unified-log errors by source, shutdown causes. Use for: app keeps crashing; Mac restarted by itself; kernel panic. | args: hours
- Collect Logs (collect_logs.sh) — Bundle crash reports + unified-log errors + install/system/wifi log tails into tar.gz on USB; sudo adds .logarchive. Use for: take evidence away; escalate with attachments; gather data for the AI to analyze. | args: hours
- Update Status (update_status.sh) — Version/build, update history, auto-update settings; --check queries Apple for pending updates. Use for: is this Mac up to date; update stuck or failing. | args: --check
- Security Check (security_status.sh) — SIP, Gatekeeper, FileVault, firewall, XProtect versions, admin users, SSH/screen/file sharing, 3rd-party extensions. Use for: security check; suspected malware; before handing the Mac back.
- User Accounts (user_account_report.sh) — Local users (uid>=500) with admin flag and last login, logged-in users, home sizes (--sizes), hidden/disabled markers. Use for: account audit; before user cleanup; who uses this Mac. | args: --sizes
- Remove Inactive Users (remove_inactive_users.sh) [DESTRUCTIVE-dry-runs-first,needs-admin] — Disable (default) or delete local users with no login in N days. Dry-run unless --force (sudo). Admin accounts are NEVER touched (no override). Also protects the current user. Use for: clean up shared Mac accounts; remove departed-user accounts. | args: days --delete --remove-home --include-unknown --exclude --force
- Find Large Files (find_large_files.sh) — Largest files (Spotlight-accelerated), folder sizes, known hogs (iOS backups, Xcode, Docker, Trash, snapshots). Use for: disk full; what is using the space. | args: path min_size_mb
- Clean Caches (clear_caches.sh) [DESTRUCTIVE-dry-runs-first] — Clean user caches, 30d+ old logs, saved app state, optional Trash. Dry-run unless --force. Use for: free space safely; weird app behavior possibly cache-related. | args: --force --trash
- Battery Health (battery_health.sh) — Cycle count, condition, capacity vs design from system_profiler + ioreg, charger info. Skips desktops. Use for: MacBook battery complaints; random shutdowns on battery.
- Full Triage Snapshot (triage_snapshot.sh) — Runs all read-only diagnostics + raw extras (ifconfig, ps, system_profiler), saves+tars to USB collections. Use for: unknown problem - collect everything; intermittent issue; gather evidence for AI analysis. | args: out_dir
- Remove Microsoft 365 (Deep) (remove_office365.sh) [DESTRUCTIVE-dry-runs-first,needs-admin] — Deep uninstall of Microsoft 365/Office for macOS: quits apps, removes app bundles + ~/Library containers/prefs/caches, forgets pkg receipts, clears keychain identity + license so products re-authenticate. Preserves Outlook mail and never touches user docs. Use for: office uninstall left files behind on mac; reinstall office clean; office signed into wrong account. | args: --force --reset-activation-only --remove-outlook-data --remove-all
- Installed Software (installed_software.sh) — Inventory installed software: /Applications with versions, Homebrew formulae/casks, and Mac App Store apps. --sort-date finds what changed recently. Use for: what's installed; did something install right before the problem; license/app audit. | args: --sort-date search-term
- Remove Adobe Creative Cloud (Deep) (remove_adobe_cc.sh) [DESTRUCTIVE-dry-runs-first,needs-admin] — Deep uninstall of Adobe Creative Cloud + apps for macOS: quits processes, removes app bundles + launch daemons/agents + ~/Library and /Library Adobe leftovers, forgets pkg receipts, clears licensing (SLStore/SLCache) + sign-in (OOBE) + keychain so it re-authenticates. Keeps Acrobat Reader; never touches creative files. Use for: adobe uninstall left files behind on mac; adobe genuine / core sync daemons after uninstall; reinstall creative cloud clean. | args: --force --remove-all --clean-hosts
- Reset Jamf Check-in (reset_jamf.sh) [DESTRUCTIVE-dry-runs-first,needs-admin] — Force a Jamf check-in/inventory/policy refresh (jamf policy/recon/manage + flush stuck policy history). Gated --reenroll removes the framework (unmanages device). Use for: jamf policy didn't run; inventory stale in jamf; mac fell out of management. | args: --force --reenroll
- Reset OneDrive (reset_onedrive.sh) [DESTRUCTIVE-dry-runs-first] — Reset OneDrive on macOS to fix stuck/failed sync, then relaunch. Files are not deleted; they re-sync. Use for: onedrive stuck syncing on mac; onedrive won't sync. | args: --force
- Clear Teams Cache (clear_teams_cache.sh) [DESTRUCTIVE-dry-runs-first] — Clear Microsoft Teams cache on macOS (classic + new Teams) to fix freezes, blank screens, login loops. Use for: teams won't load on mac; teams blank screen; teams stuck signing in. | args: --force
- Repair Network (repair_network.sh) [DESTRUCTIVE-dry-runs-first,needs-admin] — One-click network repair for macOS: flush DNS, renew DHCP, toggle Wi-Fi, clear ARP. Use for: no internet on mac; wifi connected but nothing loads; flaky connection. | args: --force --renew-only
- Encryption & Recovery Key (encryption_status.sh) — FileVault status, enabled unlock users, recovery-key posture (personal/institutional), and APFS volume encryption. Use for: is FileVault on; before wiping a Mac; who can unlock the disk.
- Reset Printing System (reset_printing.sh) [DESTRUCTIVE-dry-runs-first,needs-admin] — Clear stuck print jobs and restart CUPS. Dry-run unless --force; --remove-printers does the full reset. Use for: print job stuck; can't print; printer offline. | args: --force --remove-printers
- Domain & Directory (domain_health.sh) — AD binding, Kerberos tickets, and time sync (clock skew breaks SSO). --repair renews Kerberos and resyncs time. Use for: enterprise SSO / AD login fails; is this Mac AD-bound; kerberos errors. | args: --repair
- Wi-Fi Diagnostics (wifi_diagnostics.sh) — Live link quality (RSSI/noise, channel, PHY mode, security, rate), radio/driver info, and saved networks. Use for: wifi slow or dropping; weak signal; won't connect to wifi.
- Pending Reboot & Uptime (pending_reboot.sh) — Uptime and reboot history, plus pending macOS updates and staged installers that need a restart. Use for: has it been restarted recently; update won't finish; try a reboot first.
- Crash & Panic Report (crash_report.sh) — Kernel panics, real app crashes (memory-pressure kills shown separately), top crashing apps, and previous shutdown causes. Use for: mac keeps restarting; kernel panic; app keeps quitting. | args: days
- Backup User Data (backup_user_data.sh) — Preview the size of a user's key folders; with --force --dest copies them via rsync. Source untouched. Use for: before wiping a Mac; save user files first; migrate to a new machine. | args: --user --dest --force --all
- Antivirus Scan (defender_scan.sh) — Defender (mdatp) status, threats and on-demand scan if installed; otherwise built-in XProtect/Gatekeeper status and recent quarantined downloads. Use for: suspected malware; run a virus scan; check protection status. | args: --force --full --update
- Startup & Persistence Audit (persistence_audit.sh) — Third-party launch agents/daemons, login items, cron, and configuration profiles; suspicious paths flagged. Use for: malware/PUP cleanup; something keeps coming back; unwanted startup items.
